# Sky-Cast 🌤️

Sky-Cast is a modern, responsive weather application built with React, Vite, and Tailwind CSS. It provides real-time weather data, forecasts, and elegant UI themes based on current weather conditions.

## 🚀 Features

- **Real-time Weather Data:** Search for any city to get precise current weather conditions.
- **Detailed Forecasts:** View hourly predictions and a 7-day extended forecast.
- **Dynamic Theming:** The application background and UI components adapt smoothly with custom day and night themes.
- **Saved Recent Searches:** Your recent city searches are saved locally, allowing you to quickly revisit them with a single click.
- **Comprehensive Details:** View essential insights such as sunset/sunrise times, humidity, wind speed, and air conditions.
- **Responsive & Modern Design:** A sleek, mobile-friendly interface inspired by glassmorphism, powered by Tailwind CSS.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite
- **Styling:** Tailwind CSS (v4)
- **Icons:** [Lucide-React](https://lucide.dev/)
- **Data Fetching:** Axios
- **State Management:** React Context API

## 📦 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sky-cast.git
   ```
2. Navigate to the project directory:
   ```bash
   cd sky-cast
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App Locally

To start the Vite development server, run:
```bash
npm run dev
```
Open `http://localhost:5173` (or the provided port) in your browser.

## 💡 Usage

- **Search:** Use the search bar in the header to find weather globally.
- **Recents:** Pick past locations from your "Recent Searches" history right below the header.
- **Themes:** Test out UI changes quickly by selecting different weather theme toggles in the header.
